# Uni Animation Curve Copy Paste Drawer

Inspector で AnimationCurve をコピー・ペーストできるようにするエディタ拡張

## 使用例

```cs
using UnityEngine;

public class Test : MonoBehaviour
{
	public AnimationCurve m_animationCurve1;
	public AnimationCurve m_animationCurve2;
}
```
